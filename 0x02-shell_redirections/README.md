# 0x02-shell_redirections

## 0. Hello World

Write a script that prints “Hello, World”, followed by a new line to the standard output.

```bash
julien@ubuntu:/tmp/h$ ./0-hello_world 
Hello, World
julien@ubuntu:/tmp/h$ ./0-hello_world | cat -e
Hello, World$
julien@ubuntu:/tmp/h$ 
```
